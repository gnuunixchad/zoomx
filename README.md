# My fork of [ZoomX](https://github.com/nprezant/zoomx) - simple zooming tool for X

## Usage

| | |
|:---|:---|
| movement | <kbd>h</kbd> <kbd>j</kbd> <kbd>k</kbd> <kbd>l</kbd> |
| zoom in | <kbd>i</kbd> |
| zoom out  | <kbd>o</kbd> |
| quit | <kbd>Esc</kbd> <kbd>q</kbd> |

## Build
```sh
make install
```
